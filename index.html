League Match Companion

A desktop overlay that provides match context when your game starts. Track your performance and make informed decisions.

Built with Overwolf SDK + Riot Games API

Real-time stats for all 10 players including items, gold, and CS differentials by lane

Match history lookup showing recent ranked performance for context

Development view with live event stream, game state monitoring, and overlay preview

Features

Match History Lookup

When your game starts, fetches recent ranked match data for all 10 players to provide context about the current game.

Champion Statistics

See games played on current champion, average KDA, and CS/min trends for each player in the match.

Performance Tracking

Track your own performance patterns across sessionsâ€”CS differentials, damage output, and positioning trends.

Post-Game Analysis

Review item build timelines, gold progression curves, and detailed match breakdowns after each game.

Heatmap Visualization

Visualize your positioning patterns across multiple games to identify areas for improvement.

Local Processing

All data is processed locally on your machine. Nothing is stored externally or shared with third parties.
